# ClearPop: No Empty Cells

## Requirements

- Given a freshly generated board for any level, should have zero cells with kind 'empty' — every cell must be 'block', 'obstacle', or 'powerup'.
- Given a board after a complete clear→gravity→refill cycle, should have zero cells with kind 'empty'.
- Given a board with obstacles destroyed by adjacency damage, should still have zero empty cells after the full cycle.
