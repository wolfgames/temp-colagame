# Epic: Nana Rose Interstitial

## Requirements

- Given a level win, should show a full-screen interstitial with Nana's portrait and the zone's reward dish before the results buttons appear.
- Given an interstitial is shown, should display the zone's dish image and a short reaction line from Nana.
- Given the player taps the interstitial, should dismiss it and show the Next Level / Main Menu buttons.
- Given a level loss, should NOT show the interstitial — go straight to results.

## Asset Placeholders

Dish images live at `/assets/interstitials/zone-<N>-dish.png` (1–10).
Nana's interstitial portrait lives at `/assets/portraits/nana-rose-happy.png`.
Both are placeholders until real assets are generated.
