Drop your custom VFX effect JSON files in this folder.

Conventions in this repo:
- Effect configs: `public/assets/vfx/effects/*.json`
- Effect textures/spritesheets referenced by `textureName`: `public/assets/vfx/`

Example:
- `public/assets/vfx/effects/custom/my-effect.json`
- `public/assets/vfx/vfx-my-effect.png`
